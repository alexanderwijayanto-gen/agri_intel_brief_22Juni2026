Stok Beras 5,2 Juta Ton

Cukup Hingga 10 Bulan Kedepan

CBP per 18 Juni: 5,2 juta ton. Cadangan cukup hingga ~10 bulan (Juli 2026–April 2027). Inflasi beras melandai 0,38%. Bulog sewa gudang tambahan 2,2 juta ton. Fokus pemerintah: jaga produksi, bukan impor — artinya input pertanian (pupuk, pestisida) tetap prioritas belanja.

81.000 Ha Terbakar, Siaga Karhutla

Rekomendasi Kampanye Domestik

🎯 Skenario: NTP Tinggi → Upselling Premium

Daya beli petani di level tertinggi 34 tahun. Ini bukan waktu untuk diskon — ini waktu untuk value selling. Petani mau bayar lebih jika ada bukti ROI.

Taktik Sales:
  • Bawa data NTP + harga gabah sebagai pembuka percakapan dengan petani.
  • Tawarkan paket "Premium Protection": fungisida sistemik + biostimulan + nutrisi mikro.
  • Demo plot di sentra komoditas kopi, cabai, dan padi — tunjukkan yield gap.

🔥 Skenario: El Nino → Mitigasi Stress

Kemarau ekstrem diprediksi Juli–September. 60% irigasi rusak. Petani butuh solusi anti-stress, bukan sekadar pupuk biasa.

Taktik Sales:
  • Front-load stok biostimulan anti-stress (asam amino, fulvic acid) ke agen.
  • Push fungisida preventif sebelum musim kemarau puncak.
  • Targetkan wilayah irigasi perpompaan baru — petani baru olah lahan = butuh herbisida pra-tumbuh.

🌱 Skenario: Program Pemerintah → Akses Langsung

Benih Rp9,95T, bongkar ratoon 54.897 ha, SISKA 1,3 juta ekor — semua butuh input pendamping.

Taktik Sales:
  • Mapping kelompok tani penerima benih & bongkar ratoon — langsung tawarkan paket input.
  • Untuk SISKA: tawarkan pupuk organik & probiotik sapi.
  • Koordinasi dengan dinas pertanian daerah untuk akses program.
